## Design & Inspiration

### Image Editing & Effects

- [Ditther](https://www.ditther.com/) — A browser-based image editor for creating dither, pixel, ASCII, halftone, and other visual effects.
- [ASCII Magic](https://www.ascii-magic.com/?show-landing=1) — A browser-based tool for turning images and videos into ASCII art, pixel art, halftone, glitch, and other visual styles.
- [ASCIInator](https://asciinator.app/) — A browser-based ASCII art generator with configurable characters, sampling, colors, and multiple export formats.
- [InkField](https://ileivoivm.github.io/inkField/) — A browser-based drawing tool with configurable ink brushes, blend modes, textures, and post-processing effects.

### Branding

- [Brand Assets Generator](https://brand-generator.stoyanov.works/) — A browser-based tool for layering procedural frames, chains, intersections, shapes, and textures over images, with PNG export.

### Typography

- [Woven Type](https://airfan003.github.io/WeavingType/) — A browser-based poster editor for creating interactive woven typography with configurable threads, colors, and exports.

### Interactive Effects

- [Particles Studio](https://particles-studio.floot.app/studio) — A browser-based editor for turning text and SVGs into interactive particle effects with HTML and iframe exports.

### Backgrounds & Textures

- [Backgrounds Supply](http://backgrounds.supply/) — A collection of backgrounds for creative and web design projects.
- [ShaderGradient](https://shadergradient.co/) — A visual tool for creating animated shader gradients for React, Framer, and Figma projects.

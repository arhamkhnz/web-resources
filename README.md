## Design & Inspiration

### Image Editing & Effects

- [Ditther](https://www.ditther.com/) — A browser-based image editor for creating dither, pixel, ASCII, halftone, and other visual effects.
- [ASCII Magic](https://www.ascii-magic.com/?show-landing=1) — A browser-based tool for turning images and videos into ASCII art, pixel art, halftone, glitch, and other visual styles.

### Backgrounds & Textures

- [Backgrounds Supply](http://backgrounds.supply/) — A collection of backgrounds for creative and web design projects.
